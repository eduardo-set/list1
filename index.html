✨ Золотой одуванчик

🔹 Также может быть куплен у странствующего торговца.
🔹 При использовании на детёныше останавливает его рост (AgeLocked: 1). Повторное использование возобновляет (AgeLocked: 0).
🔹 Испускает зелёные частицы: опускающиеся (остановка) или поднимающиеся (возобновление).
🔹 Не работает на монстрах (кроме хоглинов) и жителях.
🔹 Можно посадить в горшок, использовать как жёлтый краситель или в загадочном рагу (эффект «Насыщенность»).

📌 Добавлены частицы pause_mob_growth и reset_mob_growth.

🖐️ Команда /swing

Новая команда, позволяющая размахивать руками сущностей.
Синтаксис: /swing <селектор сущности> <основная рука|вторая рука>
🔹 Работает даже с манекенами.
🔹 Выводит количество сущностей, которые начали анимацию.

💡 Атрибуты окружения

minecraft:visual/block_light_tint

Цвет освещения от блоков. По умолчанию — #FFD88C (жёлтоватый оттенок факелов).

minecraft:visual/ambient_light_color

Окружающий свет в полной темноте. Значения: Верхний мир #0A0A0A, Незер #302821, Энд #3f473f.

minecraft:visual/night_vision_color

Цвет эффекта «Ночное зрение». По умолчанию #999999.

⏰ Мировые часы и временные метки

Команда /time полностью переработана. Теперь можно создавать собственные мировые часы (world clocks) в data/<namespace>/world_clock/.

/time [of <clock>] set <time>
/time [of <clock>] pause | resume
/time [of <clock>] rate <rate> (множитель скорости)
/time [of <clock>] query time

Временные метки (time markers): minecraft:wake_up_from_sleep, minecraft:roll_village_siege и другие.

📊 Сделки, управляемые данными

Сделки жителей и странствующих торговцев теперь полностью настраиваются через JSON в папках trade_set и villager_trade.

{ "wants": { "id": "minecraft:emerald" }, "additional_wants": { "id": "minecraft:book" }, "gives": { "id": "minecraft:enchanted_book", "count": 1 }, "max_uses": 12, "xp": 1, "merchant_predicate": { ... } }

🔹 Можно указывать given_item_modifiers, предикаты, теги чар, влияющие на цену (#double_trade_price).

🏷️ Новые теги блоков, предметов и сущностей

Теги для растительности:

#supports_vegetation #supports_crops #supports_sugar_cane #supports_cactus #grows_crops #enables_bubble_column_drag_down #enables_bubble_column_push_up

Теги для предметов:

#metal_nuggets #dyes #loom_dyes #cat_collar_dyes #wolf_collar_dyes

Тег для сущностей:

#cannot_be_age_locked — мобы, на которых не действует золотой одуванчик.

Теги сделок:

#<профессия>/level_<уровень>, #wandering_trader/buying, #wandering_trader/common и другие.

🐣 Детёныши: новые модели, текстуры и звуки

Полностью обновлены модели и текстуры детёнышей всех мобов (кроме гастят и наутилусов).
🔹 Добавлены звуковые вариации для котят, волчат, поросят, жеребят, цыплят.
🔹 Седло и броня больше не отображаются на детёнышах свиней, верблюдов, волков.
🔹 Дети жителей и зомби-жителей получили слоистую модель поверх базовой.
🔹 Новые звуковые события: entity.baby_cat.ambient, entity.baby_wolf.growl, entity.baby_horse.eat и многие другие.
🔹 Добавлены звуковые вариации для взрослых: cat_royal, pig_mini/big, cow_moody, chicken_picky.

📁 Все текстуры детёнышей находятся в папках entity/.../baby/ и переименованы согласно новой структуре.

⚙️ Технические изменения и улучшения

📁 Структура мира (level.dat)

  • Тег player заменён на singleplayer_uuid.
  • Настройки сложности — строковые: peaceful, easy, normal, hard.
  • Данные битвы с драконом, погоды, правил, карт перемещены в data/minecraft/.
  • Папки измерений: dimensions/minecraft/overworld, the_nether, the_end.
  • Данные игроков: players/advancements, players/data, players/stats.

🔄 Рецепты

Добавлены новые типы рецептов: minecraft:crafting_dye, minecraft:crafting_imbue, minecraft:crafting_transmute с расширенными настройками. Удалены устаревшие специальные рецепты (карт, клонирования книг и т.д.).

🎨 Модели предметов

Поле transformation добавлено во все типы моделей. Новые специальные модели: minecraft:bell, minecraft:book, minecraft:end_cube. Модели кроватей, флагов, сундуков теперь используют part, attachment, chest_type.

🧩 Компоненты данных

🔸 minecraft:dye — цвет красителя (white, orange, magenta, …).
🔸 minecraft:additional_trade_cost — временная модификация цены в сделках.
🔸 minecraft:provides_banner_patterns теперь принимает список ID.

📈 Поставщики чисел и предикаты

Добавлены minecraft:environment_attribute, minecraft:sum, предикат minecraft:environment_attribute_check, поставщик целых чисел trapezoid.

🖥️ Системные изменения

  • Требуется Java 25 (Microsoft OpenJDK 25).
  • По умолчанию выделяется 4 ГБ RAM, сборщик мусора ZGC (для совместимых устройств).
  • Новый отладочный рендерер карты освещения (F3+4).
  • Опция «Эксклюзивный полноэкранный режим» (требует перезапуска).
  • Улучшена поддержка IME (редакторов ввода) под Windows/macOS.

🔊 Звуки трубы для музыкального блока

На медных блоках музыкальный блок играет звуки трубы, меняющиеся в зависимости от стадии окисления: block.note_block.trumpet, ..._exposed, ..._weathered, ..._oxidized.
